This machine learning competition is like no other. Teams will build Python programs that perform virtual trades on a given set of real stocks using a combination of ML and non-ML powered techniques.

  • A training set consisting of historical data for a large number of real stocks will be provided to teams.
  • Teams are expected to build a price-predicting machine learning model based on this data.
  • All teams’ programs will be given the same initial amount of virtual money to ‘invest’.
  • The program will be fed the following data multiple times:
    • New market data for that set of stocks
    • The current holdings of the program
    • The program’s available cash on hand
  • The program will be expected to make profit-maximising trades based on this data, deciding what stocks to buy or sell and in what amount.
