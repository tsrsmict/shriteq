In this event, participants will build a static site based on the requirements laid out by a real-world organisation.

Teams will be invited to The Shri Ram School, Moulsari on the morning of the 10th of October. They will be given the project requirements and will work till 3 PM that day creating a website that fits the needs of the organisation.

Rules

  • Participants are allowed to create websites using any of the following:
    • Static HTML, CSS and JavaScript
    • Site builder tools such as Wix and Squarespace
    • Wordpress
  • Participants will be allowed to use templates and 3rd-party frameworks to assist them in the creation of the site.
  • Final submissions must be in the form of a live site.
  • If using an online tool, teams must add the email of the organisers as a collaborator on the project. If they are writing code, they must provide access to a public github repository or give access to a shared folder containing the project code.

Submissions will be judged anonymously on the following criteria:

  • Success in achieving requirements of organisation
  • Design and visual appeal
  • Logical flow and structure
